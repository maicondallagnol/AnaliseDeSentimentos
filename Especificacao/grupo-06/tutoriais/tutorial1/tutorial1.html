Objetivo

Este tutorial tem como objetivo apresentar uma descrição de um sistema de análise de sentimentos utilizando a ferramenta de Aprendizado de máquina Weka. Para isto serão feitas chamadas de um sistema em Java para o programa em si, no qual serão executados os procedimentos do aprendizado.

Como objetivo secundário deste tutorial é dar suporte para a elaboração do Trabalho final da disciplina. Uma vez que este utiliza-se dos conceitos aqui apresentados para o desenvolvimento de um sistema completo de reconhecimento de sentimentos.

O que é uma Análise de Sentimentos?

Análise de sentimentos é uma sub-área do Processamento de Linguagem Natural (PLN ou NLP para sigla em inglês) na qual busca-se automatizar, através de técnicas de inteligência artificial, a detecção de sentimentos em frases, textos, entre outros dados de linguagem natural.

Para esta automatização há uma vasta gama de algoritmos e técnicas de Aprendizado de máquina que buscam solucionar este problema, como exemplo o algoritmo Naive Bayes ou as SVMs .

Para a execução de um guia prático passando pelas principais etapas realizadas numa automatização deste tipo serão utilizados códigos desenvolvidos na linguagem Java nos quais serão feito chamadas para funções implementadas na ferramenta Weka que por sua vez as executará e devolverá os resultados da análise para serem avaliados.


Criando o projeto

Para este tutorial valor utilizar apenas Java com a dependencia do jar do Weka. Para isso, vá ao meu "File" -> "New" -> "Project".

Uma janela se abrirá e nela selecione "Java" no menu ao lado esquerdo

Depois, na seção ao lado direito, selecione no menu de "Project SDK" a versão do Java 8 e clique em "Next".

Nas aba seguinte aperte "Next". Na próxima selecione o nome do Projeto e a sua pasta.

Clique em "Finish" e seu projeto será gerado.


Adicionando as dependências

Uma vez que o projeto foi criado é necessário adicionar as dependências dele, no caso o Weka.

Antes de tudo mova o arquivo weka.jar baixado para a pasta /src/ do projeto.

Para adicioná-lo ao projeto, com o projeto aberto no IntelliJ, vá em "File" -> "Project Structure".

Na janela que se abrirá ao lado esquedo no menu "Project Settings" selecione "Modules"

Na barra lateral ao lado direito da janela clique no simbolo "+"

Na mini janela que abrir clique em "JARs or directories..."

Na nova janela que abrir selecione o weka.jar seja movido para o /src/ e clique em "OK"

Com isto voltará para a janela anterior e clique em "Apply" e depois em "OK"

O conteudo de Analise.iml (no caso Analise pois este é o nome dado ao meu projeto, no caso do seu ficará de acordo com o dado) do projeto deve ser próximo a isto. Como é possível visualizar o módulo do weka.jar está presente portanto espera-se que funcione.
